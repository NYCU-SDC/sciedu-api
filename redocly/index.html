Chat

Completions_create

Generates a response based on the full conversation history provided. Returns a Server-Sent Events (SSE) stream.

Request Body schema: application/json
required
required
Array of objects (Chat.ChatMessage) non-empty

The history of the conversation, including the latest user message. The client is responsible for maintaining this context.

stream
boolean
Default: true

Whether to stream the response using Server-Sent Events (SSE). When true, the response is streamed; when false, a complete response is returned.

Responses

SciEdu server endpoint

https://dev.sciedu.sdc.nycu.club/api/chat/completions

Request samples

  • Payload
Content type
application/json
{
  • "messages": [
    • {
      • "role": "user",
      • "content": "string"
      }
    ],
  • "stream": true
}

Response samples

  • 400
  • 401
Content type
application/json
{
  • "type": "string",
  • "title": "string",
  • "status": 0,
  • "detail": "string",
  • "instance": "string"
}
